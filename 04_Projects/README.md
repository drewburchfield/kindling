# Projects

**Purpose:** Time-bound work with specific outcomes and deadlines.

**Examples:**
- Job search with target start date
- Writing portfolio creation
- Research project with deliverable

**How to use:**
- Create a folder per project
- Link heavily to permanent notes in `02_Permanent/`
- Move to `07_Archive/` when completed

**Workflow:**
1. Use `/recall` to find relevant insights
2. Build project plan linking to permanent notes
3. Create output synthesizing your knowledge
