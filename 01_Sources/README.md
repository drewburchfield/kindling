# Sources

**Purpose:** Source materials you're learning from. Books, articles, videos, writing exercises.

**Subfolders:**
- `Books/` - Book notes with metadata
- `Articles/` - Article summaries and annotations
- `Writing_Exercises/` - Practice writing with prompts

**What goes here:**
- Notes *about* a source (not your original insights)
- Highlights, quotes, summaries
- Attribution and bibliographic info

**What does NOT go here:**
- Your original insights (those go to `02_Permanent/`)
- AI-extracted content (goes to `AI_Extracted_Notes/`)

**Template:** See `09_Meta/Templates/Source-Note-Template.md`
