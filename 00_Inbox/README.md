# Inbox

**Purpose:** Universal capture. Dump everything here, sort later.

Flat structure, no subfolders needed. This is your cognitive junk drawer.

**What goes here:**
- Random thoughts and fleeting ideas
- Articles to read later
- Meeting notes to process
- Anything you haven't decided what to do with yet

**Processing:** Run `/process-inbox` to triage items one at a time.

**Goal:** Empty this regularly. Most items go to Trash (that's fine). Valuable insights get extracted to `AI_Extracted_Notes/` for review.
