# Permanent Notes

**Purpose:** Atomic insights in your own words. One idea per note. This is the core of the knowledge graph.

**Format:**
- Title states the insight (e.g., "Dopamine drives wanting, not liking")
- 50-300 words
- Your voice, not copy-paste
- YYYYMMDD prefix (e.g., `20241216-Title.md`)
- Links to related permanent notes via `[[wikilinks]]`
- Source attribution if applicable

**What goes here:**
- Your original insights and observations
- Paraphrased ideas from sources (in your words)
- Validated notes promoted from `AI_Extracted_Notes/`

**What does NOT go here:**
- AI-generated content (goes to `AI_Extracted_Notes/` first)
- Source summaries (go to `01_Sources/`)
- Project work (goes to `04_Projects/`)

**Organization:** Flat. No subfolders. Use tags for categorization, MOCs for navigation, links for structure.

**After creating a note:** Run `/find-connections <note>` to discover what it connects to.

**Template:** See `09_Meta/Templates/Permanent-Note-Template.md`

**Quality check:** Is it atomic? Is it in your words? Would it make sense a year from now without context?
