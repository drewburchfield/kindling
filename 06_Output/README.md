# Output

**Purpose:** Synthesized work created from your permanent notes. Articles, frameworks, published pieces.

**Subfolders:**
- `Articles/` - Blog posts, essays, long-form writing
- `Drafts/` - Work in progress
- `Frameworks/` - Original models and mental frameworks

**What goes here:**
- Content that synthesizes multiple permanent notes
- Published or ready-to-publish work
- Original frameworks you've developed

**Workflow:**
1. Use `/recall` to gather relevant permanent notes
2. Use `/thinking-partner` to explore your angle
3. Write the piece
4. Run `/de-ai-ify` if AI helped with drafting
