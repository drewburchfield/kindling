# Maps of Content (MOCs)

**Purpose:** Navigation hubs that organize permanent notes by theme. Think of these as curated indexes.

**When to create one:** When you have 10+ permanent notes on a related theme and need a way to navigate them.

**Format:**
- Group related notes under subheadings
- Brief context for each link (why it's here)
- Cross-reference other MOCs
- List key sources

**What goes here:**
- Thematic navigation pages
- Topic overviews that link to permanent notes

**What does NOT go here:**
- Original insights (go to `02_Permanent/`)
- Content that doesn't link to multiple notes
